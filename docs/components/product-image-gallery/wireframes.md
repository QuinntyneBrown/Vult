## Spacing Specifications

### Desktop
- Thumbnail width: 70px
- Thumbnail height: 70px
- Thumbnail gap: 8px
- Gap between thumbnails and main image: 16px
- Main image width: 535px
- Main image height: 669px
- Aspect ratio: 1:1.25

### Mobile
- Thumbnail width: 125px
- Thumbnail height: 125px
- Thumbnail gap: 8px
- Horizontal scroll padding: 16px
- Dot indicator size: 8px
- Dot indicator gap: 8px
- Dot indicator margin-top: 16px
