## Scroll Behavior

### Scroll Snap Alignment

```
Before scroll:
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│  │ ITEM 1 │ ITEM 2 │ ITEM 3 │ ITEM 4 │ ITEM 5 │ ITEM 6 │    │
│  ▲                                                           │
│  snap: start                                                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘

After clicking "next":
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│  │ ITEM 2 │ ITEM 3 │ ITEM 4 │ ITEM 5 │ ITEM 6 │ ITEM 7 │    │
│  ▲                                                           │
│  Scrolls by 1 item, snaps to ITEM 2                         │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### Momentum Scroll (Touch)

```
User swipes left ──────────────────────►

  ┌────────────────────────────────────┐
  │ Fast swipe = further scroll        │
  │ Slow swipe = shorter scroll        │
  │ Always snaps to nearest item       │
  └────────────────────────────────────┘
```
